About Dream Tech Liberia

Dream Tech Liberia is an organization that helps young people learn important tech skills. We offer courses in web development, where students learn to build websites using tools like HTML, CSS, and JavaScript. We also teach Python game development, where students create their own video games while learning how to code. Lastly, we offer Unity game development courses, where students can design 2D and 3D games using a powerful game-making tool called Unity. Our goal is to help kids and teens in Liberia learn new skills so they can create amazing things and have bright futures in technology.

We also offer bootcamp for students at verail high school.
